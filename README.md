## Issues Encountered & How I Solved Them

### Finding the Cursor editor interface

I had not used Cursor in a while (I switched to Antigravity) — and the interface had changed since I last used it. When I opened Cursor, I couldn't find the classic IDE/editor view. The website and the app seemed to surface only the **agent mode**, and I initially couldn't locate where the normal code editor lived.

**Solution:** After looking around, I found the **Editor Windows** option on the top right. Once I opened that, I had the full IDE interface back and was able to proceed with the rest of the setup.

No other issues came up while completing the process.
